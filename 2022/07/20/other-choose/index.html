近期學習計畫規劃

時間差不多在 2021 年的下半年，我在三、四個月內從 0 開始學習 Flutter 並獨立開發出畢製專題(Liver)，更加確定自己對開發 Mobile App 的熱情，也想透過 Mobile App 開發的技能來當作未來求職的主力，不過 Flutter 終究只是一個框架，雖然擁有跨平台的優點，但效能以及可擴充開發性還是不及 Native language 如 Kotlin、Swift，因此想在 Flutter 已有一定基礎的情況下，另外學習 Kotlin 並且實作一個 Side project 當作練習。

而學習 Kotlin，我預計觀看 Google Android for Developers 的

  • Kotlin Bootcamp for Programmers
  • Advanced Android in Kotlin

來當作對於 Kotlin 的基本認識，剩下的細節就在 Side project 中邊做邊摸索學習。

小孩才做選擇，我全都要！

但距離上次開發 Flutter 也有點時間了，還是必須要把它熟悉回來，所以預計會在開發完 Kotlin 的 Side Project 後，再運用 Flutter 開發另一款 Side Project，希望到時候我的創意還足夠我想出 Side Project 的點子，XD。
