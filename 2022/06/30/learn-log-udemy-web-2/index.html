CSS: Cascading Style Sheets

Syntax

change all h1’s color: h1 {color: green;}

Position

  • Inline Styling
    <h1 style="color: green;"></h1>
  • Internal Styling
    in HTML’s Head <style></style>
    write h1{color: green;}
  • External Styling
    add new file name xxx.css and write your css code
    in HTML’s Head input <link rel="stylesheet" href="./xxx.css" />

Comment
/*Comment*/ or ctrl+/

Selector

  • Universal Selector
    *{}: all element
  • Element Selector
    h1{}: all h1 element
  • Id Selector
    #idName{}: all element id is idName
  • Class Selector
    .className{}: all element class is className
  • Grouping Selector
    h1,h2{}: all h1 & h2 element
  • Descendant Selector
    div.link1 a{}: in div which class is link1 a element
  • Attribute Selector
    input[type="text"]{}: all input element which type is text
  • Psuedo Class==
    textarea:hover{}: when user’s mouse is hovers over it
  • Psuedo Elements
    p::before{content:">>";}:add at the beginning of each p element “>>”

Rule

  • Cascading
  • Specificity
  • Inheritance

Box Model

All HTML elements are box and all have this 6 attribute:
Width, Height, Content, Padding, Border, Margin

Relative Unit

  • %: The % of parent’s height/width
  • vw: The % of viewport width
  • vh: The % of viewport height

Display

Block vs Inline

  • Block’s width is default 100%
  • Inline’s width and height can’t setting

can use display:inline/block change its attribute

Inline-block

  • in webpage typesetting: inline
  • in width & height setting: block

Position

position:__;

  • Static: default(normal flow:left to right, top to down)
  • Relative: relative itself (top,left,right,bottom,z-index)
  • Absolute: relative viewport (top,left,right,bottom)
  • Fixed: always fixed on viewport
  • Sticky

Cursor

cursor: __ ;
set your mouse when it on that element

Transition

transition: __;
transition effect

  • Syntax
    transition: property name + duration + timing function
    transition: all 3s ease-in;

Transform
transform:__;

  • Translate
    transform: translate(xx px, xx px);
  • Rotate
    transform: rotate(xx deg);
  • Scale
    transform: scale(width,height);
